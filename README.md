A responsive news application built using HTML, CSS, JavaScript, and React that fetches and displays the latest news articles from various sources in real time. The app provides a clean and user-friendly interface for browsing top headlines across different categories.

 <h1>Features</h1>
 
Displays latest news headlines
Category-based news filtering (e.g., Education,Crime,Economics, Sports, Technology)
Responsive design for mobile and desktop
Fast and dynamic UI using React
Real-time data fetching using a news API

<h1>Tech Stack</h1>

Frontend: HTML, CSS, JavaScript
Framework: React
API: News API (or any external news API)
