A responsive news application built using HTML, CSS, JavaScript, and React that fetches and displays the latest news articles from various sources in real time. The app provides a clean and user-friendly interface for browsing top headlines across different categories.

 <h1>Features</h1>
 
Displays latest news headlines</br>
Category-based news filtering (e.g., Education,Crime,Economics, Sports, Technology)</br>
Responsive design for mobile and desktop</br>
Fast and dynamic UI using React</br>
Real-time data fetching using a news API</br>

<h1>Tech Stack</h1></br>

Frontend: HTML, CSS, JavaScript</br>
Framework: React</br>
API: News API (or any external news API)</br>
